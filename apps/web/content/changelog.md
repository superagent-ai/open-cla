## May 2026 · Initial release

OpenCLA is now available as an open source CLA management tool for GitHub repositories.

Set up the GitHub App, attach CLA templates to repositories, and let contributors sign personal or organization agreements directly from pull request checks.

### Included

- GitHub App checks for pull requests.
- Global CLA templates with repository-level assignment.
- Personal and organization CLA signing flows.
- Admin dashboard for repositories, templates, and signature status.
